# Quintkard UI

Slim Next.js UI shell for Quintkard.

## Run

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000` by default.

## Notes

- Built with the Next.js App Router.
- Designed as a lean shell until the Spring API endpoints are restored.
- Links currently point at the local backend on `http://localhost:8080`.
